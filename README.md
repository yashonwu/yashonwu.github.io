# Academic Profile Template

A clean academic profile template in a single `index.html` file, featuring a structured header with bio and profile photo, sectioned content cards, and a floating right-side navigation bar with smooth scrolling.

Highlights
- Biography block with right-aligned portrait, justified text, and bullet-point research areas.
- Timeline-style layouts for Work Experience, Education, Patents, Publications, and Honors.
- Fixed navigation with top/bottom jump buttons and anchored section links.
- Responsive layout adjustments for mobile screens.
